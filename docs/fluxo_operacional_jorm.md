# Fluxo Operacional JORM

## 1. Entrada do Veículo

O processo inicia quando um veículo é avaliado para compra.

Etapas:
- Avaliação do veículo
- Análise documental
- Negociação com proprietário
- Aprovação da compra
- Registro no sistema

Status:
Em avaliação → Comprado

---

## 2. Preparação do Veículo

Após a compra, o veículo entra em preparação.

Controle:
- Mecânica
- Funilaria
- Pintura
- Higienização
- Documentação

Todos os custos devem ser registrados.

Status:
Comprado → Em preparação

---

## 3. Estoque e Comercialização

Após preparado, o veículo fica disponível para venda.

Informações:
- Fotos
- Descrição
- Preço de venda
- Custo total
- Margem esperada

Status:
Em preparação → Disponível para venda

---

## 4. Venda do Veículo

Processo comercial:

- Atendimento do cliente
- Proposta
- Negociação
- Aprovação
- Contrato
- Pagamento
- Entrega

Status:
Disponível para venda → Vendido

---

## 5. Histórico

O sistema deve registrar:

- Usuário responsável
- Data e hora
- Alterações realizadas
- Custos lançados
- Mudanças de status
- Vendas realizadas
